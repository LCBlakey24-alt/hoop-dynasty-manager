# Hoop Dynasty Manager Data Notes

## Current Team Data

The app currently includes all 12 planned British Super Basketball League teams.

Each team currently has a 10-player roster:

- 5 starters
- 3 bench rotation players
- 1 veteran/depth player
- 1 young prospect

This gives the project enough data to build the dashboard, team selection, roster screen, early match simulation, and basic rotation/tactics features.

## Current Player Fields

Players currently include:

- Name
- Age
- Position
- Role
- Archetype
- Overall
- Potential
- Morale
- Form

## Current Player Roles

- Starter
- Rotation
- Depth
- Prospect

# Hoop Dynasty Manager

A fictional basketball management game prototype.

## Vision

Hoop Dynasty Manager is a web-based basketball management game where players build a fictional franchise, manage rosters, set tactics, simulate seasons, develop talent, and eventually experience a dedicated player career mode.

## Early Direction

- Fictional teams and players first
- Web-based prototype
- Manager mode as the first playable loop
- Player career mode planned after the match/season engine works
- Dark sports command-centre UI using the Midnight Court theme

## First Playable Target

The first target is a small but working loop:

1. Pick a fictional team
2. View the roster
3. Set a starting five
4. Simulate the next match
5. View the result and box score
6. Update the league table
7. Progress through a season

Working title: **Hoop Dynasty Manager**
